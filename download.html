Quick start

  1. Unzip the beta packet and run Orrery.exe. Windows SmartScreen may warn on this portable beta — choose More info → Run anyway.
  2. Orrery opens the Agents/Nexus desktop shell with a seeded sandbox workspace — no other installs required.
  3. Optional local route: install Ollama and pull a model. Running local/BYOK routes requires a subscription or legacy access.
  4. Or sign in from Settings -> Account and subscribe - hosted DeepSeek API, Doubleword, and Arbiter credits work with zero provider-key setup.
  5. Checksums, manifest, and the package security report live on the binary release page.
