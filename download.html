Quick start

  1. Unzip the bundle and open INSTALL.md.
  2. Install Node 20+, pnpm, and (optional) Ollama for local models.
  3. Run start-orrery.ps1 (Windows) or start-orrery.sh (macOS/Linux).
  4. Open the printed URL — sidecar on ws://127.0.0.1:4575, web UI on port 5173.
